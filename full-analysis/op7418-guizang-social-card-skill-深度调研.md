## 一、项目定位

Guizang Social Card Skill 是一个适配 Claude Code / Codex 环境的**图文卡片技能**，专为小红书（Rednote）图文、公众号封面对等社交平台内容生成而设计。它不需要设计师，Agent 根据文字即可生成完整的高质量图文组件，输出 PNG 图片。

### 为什么要解决这个问题
- 小红书图文和公众号封面对是中文社交内容创作者的"刚需"
- 传统工具（Canva/PS）门槛高、无法嵌入 Agent 工作流
- Agent 可以写作但不会排版——这个 Skill 填补了这个缺口

## 二、双视觉系统

项目核心是两套独立但共用的视觉系统：

### 1. 电子杂志风（Editorial）
- **灵感来源**: Monocle / Kinfolk / Cereal 风格的克制版面
- **适用**: 叙事、生活方式、旅行、阅读、影视、个人观察
- **16 个版式骨架**: M01-M16，覆盖封面（Image-Led Cover）、Pipeline、Before/After 等
- **6 套主题**: 墨水经典、靛蓝瓷、森林墨、牛皮纸、沙丘、午夜墨（暗色）

### 2. 瑞士国际主义（Swiss）
- **设计原则**: 网格系统、单一锚点色、直角发丝线、极致字号对比
- **适用**: 产品测评、数据、方法论、教程、AI 工具
- **12 个版式骨架**: S01-S12，覆盖 KPI Tower、H-Bar Chart、Matrix + Hero 等
- **4 套锚点色**: 克莱因蓝（IKB）、柠檬黄、柠檬绿、安全橙

## 三、技术架构

### 输出管道

```
文章/文案 → Agent 分析 → 选版式 → 取图源 → Compose HTML → Playwright 渲染 → PNG
```

### 关键设计

1. **单文件 HTML 渲染**：所有图文组件输出为单 HTML 文件，Agent 可读写修改
2. **Playwright 渲染**: `node render.mjs` → 直接生成 PNG，无需构建链
3. **图源工作流**: 多级优先级（用户图 > Unsplash > Pexels > Flickr CC > Wallhaven > 直接搜索），自动写 `SOURCES.md`
4. **校验脚本** `validate-social-deck.mjs`：6 条硬规则（溢出检查、字号上限、4 横带密度、footer 碰撞等）

### 三个画板尺寸

| 命令 | 尺寸 | 用途 |
|------|------|------|
| `.poster.xhs` | 1080×1440 | 小红书 3:4 图文 |
| `.poster.wide` | 2100×900 | 公众号 21:9 头图 |
| `.poster.square` | 1080×1080 | 公众号 1:1 分享卡 |

## 四、AI 友好设计

### 7 步自动化工作流
1. **Intake** — 抓取目标平台、风格、内容素材、用户图
2. **Style & Theme** — 选 Editorial 或 Swiss + 主题色
3. **Layout Selection** — 从 28 个版式中挑选适配的内容布局
4. **Asset Prep** — 多源取图 + 自动记录来源
5. **Compose & Render** — 组装 HTML → Playwright 渲染
6. **Deliver & Review** — 先给用户看图，默认不自动验证
7. **Iterate** — 反馈循环：改 inline 样式、swap 版式、重渲

### Agent 触发关键词
- "帮我做一套小红书图文"
- "做一个公众号 21:9 头图 + 1:1 分享卡"
- "生成一套瑞士风社交卡片"

## 六、关键优势与不足

### 优势
- 🔥 **3.2K Stars 的速度**：推出后迅速获得关注，证明中文社交内容 Agent 化需求强劲
- 🎨 **双视觉系统**：Editorial 做氛围叙事、Swiss 做事实结构，满足不同内容类型
- 🤖 **Agent 原生设计**：从触发方式到工作流都围绕 Agent 能力设计
- ✅ **可校验质量**: 6 条真实 DOM 测量规则，不是静态扫描
- 🌏 **图文采集团**: 多源 + 来源标记，符合版权规范

### 不足
- 📡 **依赖 Playwright**: 用户需安装 Node.js 和 Playwright 渲染环境
- 🖼️ **网络取图不确定**: 取图成功率和速度受网络影响
- 🔒 **不允许自定义颜色**: 主题色预设限定，用户无法微调
