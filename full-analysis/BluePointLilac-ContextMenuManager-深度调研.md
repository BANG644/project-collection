# BluePointLilac/ContextMenuManager 深度调研报告

> 调研日期：2026-06-15 | 语言：C# (.NET 3.5/4.0) | 协议：GPL-3.0

## 一、项目定位

**ContextMenuManager** 是一款纯粹的 Windows 右键菜单管理程序，由个人开发者"蓝点 lilac"独立开发。它提供图形化界面，让用户轻松管理 Windows 资源管理器中的各种右键菜单项目，无需手动操作注册表。

### 核心定位
- **右键菜单的全方位管理**：启用/禁用、修改名称/图标、导航注册表位置、永久删除
- **纯净安全**：使用系统提供的键值进行隐藏，而非暴力备份删除法
- **面向所有 Windows 用户**：从电脑小白到高级用户都能使用

## 三、功能全景

### 右键菜单场景覆盖
| 场景 | 说明 |
|------|------|
| 文件 | 右键点击文件时的菜单 |
| 文件夹 | 右键点击文件夹时的菜单 |
| 新建 | "新建"子菜单 |
| 发送到 | "发送到"菜单 |
| 打开方式 | 文件关联菜单 |
| 自定义文件格式 | 特定文件类型的菜单 |
| IE 浏览器 | Internet Explorer 菜单 |
| WinX | Win+X 快捷菜单 |

### 操作能力
- **启用/禁用** — 安全隐藏不需要的菜单项
- **修改名称** — 自定义菜单显示名称
- **修改图标** — 更换菜单图标
- **导航注册表位置** — 快速定位菜单项对应的注册表路径
- **导航文件位置** — 快速定位关联的文件路径
- **永久删除** — 完全移除菜单项（谨慎操作）
- **自定义添加** — 添加自定义菜单项目

## 五、社区口碑

### 优势
- **实用性极强**：Windows 用户高频刚需工具
- **操作安全**：使用注册表隐藏而非删除，避免系统损坏
- **开源免费**：GPL-3.0 协议，完全免费且无广告
- **轻量纯净**：单 exe 文件运行，无需安装

### 局限
- **个人开发**：更新频率受限于个人开发者时间
- **仅限 Windows**：不支持 macOS/Linux
- **误报问题**：对注册表的操作可能被 Windows Defender 误报为病毒
- **功能范围有限**：仅管理右键菜单，不涉及其他系统设置

## 七、核心研判

1. **Windows 工具的长尾需求**：右键菜单管理是 Windows 用户的经典需求，ContextMenuManager 以"纯粹"的定位在这个细分领域站稳了脚跟
2. **安全优先的设计哲学**：不同于竞品的暴力备份删除法，使用注册表键值隐藏是更安全的设计选择
3. **个人开源项目的典范**：虽然由个人开发者维护，但代码质量、功能完整度和社区反馈都达到了较高水平
4. **中文社区的重要开源项目**：作为中文开发者个人项目，获得了大量中文用户的认可和支持
