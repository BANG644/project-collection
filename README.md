## 🔍 关于调研标准

每份深度调研报告包含以下维度：

1. **项目定位** — 一句话定位 + 解决的核心问题
2. **项目全景** — Stars/Forks/语言/许可证/主题等基础指标
3. **核心架构** — 目录结构分析 + 技术栈判断 + 关键文件精读
4. **源码深度解读** — 入口文件、配置分析、设计亮点
5. **社区口碑** — Issues/PRs/Releases 抽样分析
6. **竞品对比** — 同类项目横向对比
7. **核心研判** — 优势/风险/适用场景/最终评分
